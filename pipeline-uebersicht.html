Redline Grow Meeting Recording Pipeline

Automatische Aufnahme, Transkription und KI Analyse aller Meetings

📅

1. Termin wird gebucht

Calendly Google Calendar Microsoft Teams

Kunde bucht über Calendly einen Termin. Calendly erstellt automatisch einen Google Calendar Eintrag mit Microsoft Teams Meeting Link. Auch manuell erstellte Kalendereinträge mit Meeting Link werden erkannt.

LIVE
💬

2. Slack Erinnerung (2 bis 5 Min vorher)

n8n Workflow Google Calendar API Slack DM

Alle 3 Minuten prüft der n8n Workflow deine nächsten Termine. Wenn ein Meeting in 2 bis 5 Minuten startet, bekommst du eine Slack DM mit Titel, Uhrzeit, Teilnehmer und direktem Teams Link zum Klicken.

LIVE
🎙️

3. Fireflies nimmt automatisch auf

Fireflies.ai Microsoft Teams

Fireflies erkennt den Termin über deinen Google Calendar und tritt automatisch als stiller Teilnehmer dem Teams Meeting bei. Der Kunde sieht nur "Fireflies.ai Notetaker" in der Teilnehmerliste. Keine Chat Nachrichten, keine Emails, keine Voice Trigger. Du fragst mündlich kurz nach Einverständnis.

LIVE
📝

4. Transkription auf Deutsch

Fireflies.ai Sprache: Deutsch

Nach dem Meeting verarbeitet Fireflies die Aufnahme und erstellt ein vollständiges deutsches Transkript mit Sprechererkennung (wer hat was gesagt). Fireflies erstellt auch eine eigene Zusammenfassung und Action Items.

LIVE
🔄

5. n8n holt Transkript ab (alle 2 Stunden)

n8n Workflow Fireflies GraphQL API

Der Fireflies Transcript Puller Workflow fragt alle 2 Stunden die Fireflies API nach neuen Transkripten. Neue Transkripte werden automatisch weitergeleitet zur GPT Analyse. Zusätzlich gibt es einen Webhook Workflow für sofortige Verarbeitung.

LIVE
🤖

6. GPT analysiert das Transkript

OpenAI GPT-4o-mini n8n Code Node

GPT liest das komplette Transkript und extrahiert: Zusammenfassung (2 bis 3 Sätze), Stimmung (positiv/neutral/negativ), Action Items, Kundenthemen, Risiken und nächste Schritte. Alles auf Deutsch, optimiert für Social Recruiting Kontext.

LIVE
💾

7. Daten werden gespeichert

n8n Data Table: Meetings n8n Data Table: Kundenintelligenz

Die Analyse wird in zwei Data Tables geschrieben. Die Meetings Tabelle bekommt Zusammenfassung, Stimmung und Action Items. Die Kundenintelligenz Tabelle bekommt pro Kunde die aktuellsten Erkenntnisse, Risiken und nächste Schritte.

LIVE
🚀

8. Slack Nachricht + Dashboard

Slack #redline-c-level meetings.html intelligence.html

Du bekommst eine Slack Nachricht in #redline-c-level mit der kompletten Analyse: Zusammenfassung, Stimmung, Action Items, nächste Schritte. Gleichzeitig erscheinen die Daten im Dashboard unter Meetings und Kundenintelligenz.

LIVE

So testest du die Pipeline jetzt

  1. Schnelltest: Buche über Calendly einen Testtermin mit einem Mitarbeiter (z.B. Justin oder Alexander) für in 10 Minuten. Microsoft Teams Link wird automatisch erstellt.
  2. Slack Reminder prüfen: 2 bis 5 Minuten vor dem Termin sollte eine Slack DM bei dir ankommen mit Titel, Uhrzeit und Teams Link.
  3. Fireflies prüfen: Wenn du dem Teams Meeting beitrittst, sollte nach ca. 30 Sekunden "Fireflies.ai Notetaker" als Teilnehmer erscheinen.
  4. Transkript prüfen: Nach dem Meeting (ca. 5 bis 15 Min Verarbeitungszeit) erscheint das Transkript in deinem Fireflies Dashboard unter app.fireflies.ai.
  5. Analyse prüfen: Spätestens beim nächsten 2h Polling Zyklus (oder sofort per Webhook) wird GPT das Transkript analysieren und du bekommst die Slack Nachricht mit Zusammenfassung.
