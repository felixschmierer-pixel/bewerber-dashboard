Schritt 1: Funnel Typ bestimmen

Wählen Sie, wie Ihre Daten strukturiert werden sollen:

Standard Beruf und Firma
z.B. "Stuckateur und Meier"
Mit Standort Beruf und Firma | Ort
z.B. "Stuckateur und Meier | München"
Mit Quelle Beruf und Firma | Quelle
z.B. "Stuckateur und Meier | LinkedIn"

Schritt 2: Webhook URL generieren

Einfach Position und Firma eingeben. Das System erkennt automatisch, welcher Teil die Firma ist. Reihenfolge und Format sind egal.

Regeln für Webhook URLs

  • Verwenden Sie "und" statt "&" in der URL
  • Der Text vor "|" muss exakt gleich sein
  • Format: Beruf und Firma | Detail
  • Keine Sonderzeichen außer Leerzeichen und Bindestrich

Schritt 3: Webhook in Meetovo eintragen

Folgen Sie diese Schritte in Meetovo:

1. Dashboard aufrufen
Gehen Sie zu Ihrem Meetovo Dashboard und wählen Sie den gewünschten Funnel
2. Einstellungen öffnen
Klicken Sie auf das Zahnrad Symbol und dann auf "Integrationen" oder "Webhooks"
3. Webhook URL einfügen
Kopieren Sie die generierte Webhook URL und fügen Sie sie in das entsprechende Feld ein
4. Speichern und aktivieren
Speichern Sie die Einstellungen und aktivieren Sie den Webhook

Schritt 4: Webhook testen

So testen Sie, ob alles funktioniert:

  • Senden Sie eine Test Bewerbung in Meetovo
  • Warten Sie 1 bis 2 Minuten
  • Aktualisieren Sie dieses Dashboard
  • Die neue Bewerbung sollte in den Kürzliche Bewerbungen angezeigt werden

Häufige Probleme

  • Keine Daten erscheinen: Überprüfen Sie die Webhook URL Syntax
  • Falsche Kategorie: Der Text vor "|" muss exakt gleich sein
  • Zahl bleibt gleich: Webhook könnte nicht aktiv sein
  • API funktioniert nicht: Überprüfen Sie die API URL oben
